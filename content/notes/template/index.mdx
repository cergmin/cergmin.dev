## KaTeX

Можно вставлять математические формулы.
Чтобы найти гипотенузу, вам нужно использовать теорему Пифагора:
$c^2 = a^2 + b^2$ или $c = \sqrt{a^2 + b ^ 2}$.
Ещё есть формула Герона,
ей можно воспользоваться для нахождения площади любого треугольника,
если вам известны длины всех сторон.
Сначала нужно найти полупериметр: $p = \frac{P}{2} = \frac{a + b + c}{2}$,
а потом и саму площадь: $S = \sqrt{p(p - a)(p - b)(p - c)}$.

Если ваша формула слишком длинная, чтобы вставлять её в текст,
то можно вывести её как отдельный блок.

$$
\begin{cases}
  \dfrac{xy^2 - 2xy - 4y + 8}{\sqrt{4 - y}} = 0 \\
  y = ax
\end{cases}
$$

## Вставляем исходный код в статью

Угу, вы не ослышались, в статьи можно вставлять код.
Есть инлайновый вариант, вот такой: `print('Hello, wold')`.

Но есть и другая, более крутая штука.
Смотрите, целый блок кода, да ещё и с подсветкой!

```js
const makeRequest = () =>
    getJSON()
        .then(data => {
            console.log(data)
            return "done"
        })

makeRequest()
```
